# README #

Práctica 2DAM Programación multimedia

Uso de PokeApi para mostrar una Pokedex

La app tiene un menú lateral con 4 opciones:

	Generaciones: Te muestra las 3 primeras generaciones de Pokemon en un contenedor de fragments en el que hay un ViewPager con 3 fragments
	Movimientos: Muestra una lista de movimientos disponible en Pokemon
	Bayas: Muestra una lista de bayas disponible en Pokemon
	Items: Muestra una lista de items disponible en Pokemon

Toda la información que se muestra en la app se obtiene de la API pokeapi mediante el uso de Retrofit
